## D. Project Scope

### In Scope

The project focuses on developing an AI-powered Investment Research Agent for NIFTY 50 companies. The following functionalities are included within the project scope:

#### Data Collection

* Stock market data ingestion
* Company news ingestion
* Automated daily data updates using Apache Airflow

#### Data Storage

* PostgreSQL for structured data storage
* Neo4j for relationship and graph-based data storage

#### Data Processing

* Sentiment analysis of company-related news
* Research score generation
* Risk score generation

#### User Interface

* Streamlit-based/react.js dashboard
* Company research view
* News intelligence view
* Risk analysis dashboard

#### Supported Companies

The initial implementation focuses on selected NIFTY 50 companies, including:

* Reliance Industries
* TCS
* Infosys
* HDFC Bank
* ICICI Bank
* Bharti Airtel
* ITC
* Larsen & Toubro
* State Bank of India
* Axis Bank

The scope can later be expanded to cover all NIFTY 50 companies.

### Out of Scope

The following features are not included in the current project scope:

* Real-time stock trading
* Buy/Sell investment recommendations
* Portfolio execution and management
* Cryptocurrency analysis
* Options and derivatives trading analysis
* High-frequency trading systems

### Expected Outcome

The project aims to provide a centralized platform that automates investment research by collecting, processing, and analyzing stock market and news data, enabling users to make more informed investment decisions.
